---

## 🎮 스테이지 구성  

### 🐡 캐릭터  
- **복어 이동**:  
  - 제자리에서 좌우 이동 및 점프 가능  
- **먹이(노트)**:  
  - 먹이를 먹으면 콤보 수 증가 및 스코어 상승  
  - 먹이를 놓치면 생명력 감소 (초기 생명력: 6개)  
- **아이템(별)**:  
  - 복어가 일시적으로 커져 2~3개의 레인을 동시에 먹을 수 있음  

### 🏆 점수 시스템  
- **스테이지 해금 조건**:  
  - 각 스테이지의 최고 점수를 누적 점수로 달성 시 다음 스테이지 해금
